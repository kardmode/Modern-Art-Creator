Creating modern art is easy! Simply specify your desired canvas size and the number of lines you wish to draw, type some text into the box, then watch as your masterpiece is created.

The drawing created is based on the settings and text specified, so the same settings and text will always produce the same drawing.

Multiple drawings added to the same image will stack on top of each other. Simply refresh the page to start a new image.
